# Speedpanel Estimator

Ready-to-go React + Vite + Tailwind project for the Speedpanel estimator.

## Run in GitHub Codespaces

```bash
npm install
npm run dev
```

Then open the forwarded port **5173** from the Codespaces **Ports** tab.

## Run locally

```bash
npm install
npm run dev
```

Open the URL Vite prints in the terminal.

## Check before release

```bash
npm run typecheck
npm run build
```

## Main files

- `src/App.tsx` — estimator app
- `src/index.css` — Tailwind entry
- `tailwind.config.js` — Tailwind scan paths
- `package.json` — scripts and dependencies

## Scripts

- `npm run dev` — start dev server using `vite --host 0.0.0.0`
- `npm run typecheck` — TypeScript check
- `npm run build` — production build
- `npm run preview` — preview production build
